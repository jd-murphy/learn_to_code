Evens Only

Write a program that takes user input until the user is finished entering numbers. The user will type in an integer number (no decimals) and press enter. If the number they enter is even, add it to a list called even_entries. Do not add odd numbers to the list. When the user is finished, they will press enter twice and the program will show the resulting list.

even_entries = []
entry = input("Enter a number: (Press enter twice to quit) ")
while entry:
  entry = int(entry)
  if entry % 2 == 0:
    even_entries.append(entry)
  entry = input("Enter a number: ")
print("even_entries")
print(even_entries)
