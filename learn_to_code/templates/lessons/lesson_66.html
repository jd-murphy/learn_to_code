Ughh, taxes...

Okay coder, we've talked about writing code for an online shopping application and cash register software. Until now, we've avoided a harsh reality... Most transactions are taxed by the government. Taxes are usually a percentage added on at the end of a transaction. Let's write a function to make our shopping application more realistic by calculating tax for the purchase. Remember we need to convert our input to a float when we work with numbers with decimals, like money!

def add_tax(total):
  tax_rate = 8.25
  tax = total * (tax_rate/100)
  return total + tax

purchase_total = float(input("Enter the merchandise total: "))
final_total = add_tax(purchase_total)
print(f"Your total including tax is ${final_total}")
