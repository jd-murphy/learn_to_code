Lesson 58

Evens only, please!

Okay coder, let's use what we learned to make sure only even numbers are in our list. If we find an odd number, let's add 1 to it and make it even! We'll use modulo % to check every number in our list. We know that every even number divided by 2 is 0. We'll have python check if each number % 2 is not equal to 0. We use != to check if something is not equal to something else. If the answer is True we know the number is odd so we add one to it to make it even, if the answer is False the number must already be even.

nums = [3, 5, 2, 6, 9, 19, 34, 23, 6, 13, 930, 234, 123, 643, 5, 353, 7564, 99, 102]
for i in range(len(nums)):
  if nums[i] % 2 != 0:
    nums[i] += 1
print(nums)

Take a look at the numbers in the list now. They are all even! This lesson makes a good example of how we need to think sometimes when we are coding. As programmers we need to challenge our ways of thinking and use logic to translate our ideas into code that python can understand.
