Another format!

Okay coder, we've seen how to .format() a string and pass in variables. But in the newest version of python that we are using (Python 3) there is an even easier way! Let's use f-string to put our data into a string! f-strings use a different syntax, and I think you'll get comfortable with it quickly! Normally we wrap our strings in quotes like this "Hey!" or this 'Howdy!' and either way is fine. With f-strings we are going to prefix our string with the letter f like this: f"Hey there!" Then, to put data in our f-string, we use curly braces {} with the variable inside as we have before. f"Hey there {name}!"

name = "Jeffrey"
print(f"Welcome, {name}!")

balance = 237.94
print(f"Welcome, {name}. Your balance is ${balance}")

There are lots of ways to format strings in python, and you can use whichever one you want. I prefer f-strings with Python 3. With earlier versions of Python you might have to use another method to format, as shown in previous lessons.
