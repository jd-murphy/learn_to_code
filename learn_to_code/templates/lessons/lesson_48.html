Lists? Lists!

Okay coder, we've got to learn about lists. In python, a list lets us store more than one piece of information in a variable. We'll even be using them with for loops! A list looks like this

names = ['Thomas', 'Rebekkah', 'Rachel', 'Aaron', 'Andre']
print(names)

We can put pretty much anything we want inside a list. str, int, float, and all kinds of other things. True and False are examples of a built in data type in python called boolean and we can store boolean data in lists too!

info = ['Jessie', 'Jiminez', 'student', 22, True, False]
print(info)

Lists are dynamic and useful and can help us create really powerful programs. In the next lesson we will start working more with lists.
