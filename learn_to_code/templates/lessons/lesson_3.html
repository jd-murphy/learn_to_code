Math can be fun?

Okay coder, Welcome to your third coding lesson!

Python can do all kinds of awesome things, even math! Let's learn how to do some simple math. In the editor, let's type our math problem in the print statement. This time, we won't use quotes " ".

print(2+7)

Make sure you keep the parentheses ( ), but don't use quotes " "! Now click the green button above the editor to run your code! Look in the console and you should see

9

Easy, huh? Run it again and try two different numbers!
