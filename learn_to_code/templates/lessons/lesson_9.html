m = "Variables rock!"

Okay coder, we've seen our first variable, but that is just the beginnning. Variables are so important to programming. Variables give you the power to move information around and do anything you want to that information. You can save it, you can change it, and you can print() it out when you need to! Python uses variables to store information. Try this

some_variable = "Data goes in variables."
print(some_variable)

Hopefully you're starting to see how awesome variables are!

This is just the beginning. We can store any string in a variable. Try putting your name in some_variable. You can have more than one variable at the same time, just make sure you declare them each on a separate line. On the line after some_variable try making a variable called another_variable and store a string message in it. Be sure to print(some_variable) and on the next line print(another_variable) to see your output!
