Lesson 55

Increase elements in a list

Okay coder, let's use what we learned to increment each element in a list by 1.

nums = [1, 2, 3, 4, 5, 6, 7]
for n in range(len(nums)):
  nums[n] += 1
print(nums)

We are using the same process as in the last lesson. We find the length of the list with len(). The length is 7 so we use 7 to create a range() from 0 to 6. Then we loop through the sequence of numbers that range() gives us: 0, 1, 2, 3, 4, 5, 6. Each iteration through the loop, n is going to be the next number from the range(). We use n as the index to access each element in the list nums. Because we used the length of nums and range() to control our loop, we don't need to worry about index errors. They both start at index 0. For each item in the list we increment it by 1.
