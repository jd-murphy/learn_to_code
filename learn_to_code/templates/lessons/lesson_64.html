Parameters and Arguments

Okay coder, we've written a simple function, but to be useful we need to make our functions dynamic. We've used python's built in print() function. You've noticed that every time we use print(), we pass in the string we want it to display in the terminal. That string is called an argument and shows how dynamic functions can be. print() does the same thing no matter what string we give it: It displays the argument in the terminal. We can set up our own functions to accept arguments. In a function's definition, we define parameters to hold the arguments we need the function to accept. When we call the function we pass in an argument for each parameter.

def do_math(a, b):
  print(f"{a} plus {b} is {a+b}")
  print(f"{a} minus {b} is {a-b}")
  print(f"{a} multiplied by {b} is {a*b}")
  if b != 0:
    print(f"{a} divided by {b} is {a/b}")

do_math(14, 6)
do_math(3, 9)
do_math(17, 8)
do_math(62, 0)
