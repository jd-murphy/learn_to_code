So sensitive!

Okay coder, we've seen how to take input from our user and give it to python. The problem is, we were checking for the specific string "programmer" and if the user typed in "Programmer" or "PROGRAMMER" we wouldn't get the right result... Of course, python has a way to solve this problem! Python is case sensitive, that means that "true" is different than "True" is different than "TRUE". So we have to make sure python can handle any capitalization of a word and treat it as the same word. The trick is to use .upper() or .lower()! .upper() and .lower() are methods we can use to change strings in python. Check it out! Type in a word in all lowercase letters and run your code.

data = input("What is the secret word? ")
print(f"The secret is -> {data.upper()}")

.lower() works just the same. Try typing in your name in proper capitalization like this: Firstname Lastname. Run the code to see how python uses .lower().

name = input("Enter your name: ")
print(f"Transformed to lowercase -> {name.lower()}")
