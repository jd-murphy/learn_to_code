Add 'em up!

Okay coder, we've seen how to do get input with the input() function, and turn the str data type into the int data tyep using the function int(). We can wrap the input() function in the int() function to make our code more organized. Let's use it to take in a few numbers from the user and add them up!

num_1 = int(input("Enter the first number: "))
num_2 = int(input("Enter the second number: "))
print(f"{num_1} plus {num_2} is {num_1 + num_2}")
