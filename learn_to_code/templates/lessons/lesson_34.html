Lesson 34

Not so sensitive now...

Okay coder, we've seen how to take some text and transform it with .upper() or .lower(). Let's apply this to our earlier example.

name = input("What is your name? ")
age = input("How old are you? ")
occupation = input("What is your job? ")
print(f"Your name is {name}, and you are {age}.")
if occupation.lower() == "programmer":
print("Programmer? Awesome! Excellent choice!")
else:
print(f"So you're a {occupation}. Interesting.")

Now that we are using .lower() to make sure our user input is always consistent in its casing, we can be sure that anyone who enters "programmer" will be greeted as a programmer, whether they type it like this, "programmer" or like this, "Programmer" or like this, "PROGRAMMER" or like this, "prOGraMmEr" or... well, you get the picture.
