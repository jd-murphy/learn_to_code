Okay coder, we've seen how to store string messages in variables, but we can store other kinds of data in variables as well. We are going to learn about more types of data soon, but we already know about numbers. Let's store some numbers in variables! If we want to do math in python we can store the result in a variable. Try this

answer = 0
answer = answer + 2
print(answer)
answer = answer + 2
print(answer)
answer = 0
print(answer)

We will learn another way to change a variable in a bit. But for now, we know we can change the value of the variable, but we can also us the variable itself to change its own value! Just like this answer = answer * answer. Try setting answer to a number that's not zero, and then use answer and multiplication * to change answer!
