If you insist...

Okay coder, we've seen how if statements can help us make decisions in our code. The fun is just beginning. Let's check out another if statement.

enjoy_coding = True
learning = True
if learning:
print("Keep it up! You're doing amazing!")
if enjoy_coding:
print("Keep studying and you can be a python expert!")

Don't forget you need to press tab on the line below the if!

In the above code, each variable is checked to see if it is True. When the if statement finds a True value, it exectutes the code inside of the if (the print on the next line). If you change either of those variables to False, the if statement would skip the print() statement for that variable.
