Let's see... What else?

Okay coder, we've seen how to use an if... but what if that's not enough? What if we need some code to run if the if statement evaluates to True and some different code to run if the if statement is False? Well it might surprise you how simple it is. if matches up with else to make that happen!

account_balance = 1475.96
purchase_total = 299.00
if account_balance > purchase_total:
  print("You have enough money. Purchase successful.")
else:
  print("You don't have enough money. Purchase cancelled.")

Don't forget to press tab after the if line and the else line. Also pay attention to your colons : at the end of those two lines! Try runnning it with different ammounts in the account balance and different purchase prices to really understand how the logic works. You can make some really complex decisons now that you have if and else.
