Out of range!

Okay coder, we know how to find out how long a list is. The last lesson said we should use len() to find out how long our lists are... But let's SEE why we need to know! Errors are really helpful to us programmers. We aren't afraid of them. We need to get familiar with them so we can fix them when something is wrong with our code. Let's try to access an element that is isn't in our list!

nums = [4, 0, 2, 17, 8]
print(len(nums))
print(nums[5])

Our list is 5 elements long. That means the highest index is 4. An index of 5 is out of range.

Traceback (most recent call last):
File "python", line 2, in <module>
IndexError: list index out of range

By reading our error we know exactly what's wrong!
