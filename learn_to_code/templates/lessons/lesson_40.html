But we need to learn about a potential problem we might run into with some loops. The problem is an infinite loop, a loop that never stops. This can be troublesome because it can crash your program and cause problems for your user. We try to avoid infinite loops. But what fun would it be if we said not to do something, and didn't do it! üòù Let's make an infinite loop and see why it's a problem!
