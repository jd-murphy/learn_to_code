Lesson 59

By threes!

Okay coder, we've seen quite a bit of new stuff. This is a good time to talk about solving problems in code. There is ALWAYS more than one way to solve a programming problem. Let's look at different ways to make a list counting by 3s from 1 to 100 using everything we've learned. To add an item to the end of a list we use .append() like this

threes = []
for i in range(1, 100):
  if i % 3 == 0:
    threes.append(i)
print(threes)

Or we could do it this way

threes = []
for i in range(3, 100, 3):
  threes.append(i)
print(threes)
