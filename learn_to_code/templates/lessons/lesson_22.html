Decisions, decisions...

Okay coder, we've got some awesome tools to work with. Now we need to learn how to make a smart computer program. And by smart, I mean make decisions. That's right. Your code can make decisions. Let's see how. The questions we have been asking python up until now are mostly is questions. Is 5 greater than result? Is current_speed less than speed_limit? These kinds of quesitions are very important in coding and we are going to keep using them. But let's add the big if into the mix. In python we can ask if something is True. Check it out! Let's check back with our video game.

lives_left = 3
if lives_left > 0:
  print("You have enough life. Keep playing!")

You need to make sure you press the tab key (or 2 spaces) on the line after your if statement. Python uses whitespace like tabs and spaces to figure out what you are trying to tell it to do.

There are some new things like the colon : at the end of the if statement line, and the whitespace on the line below the if. But all in all, it's really just a simple question. And python is really happy to give us an answer!
