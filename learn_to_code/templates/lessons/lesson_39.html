A second look

Okay coder, let's look at the code from the last lesson. That was a lot of new stuff.

total = 0.0
 price = input("Enter the price of the item: ")
 while price:
   total += float(price)
  price = input("Enter the price of the next item: ")

print(f"Your total is ${total}")
1 / 6

The first line declares and initializes a variable called total to keep track of the total price.

2 / 6

The next line starts getting user input, which we're familiar with. We store the input into the variable price.

3 / 6

On the next line we start our while loop. Loops repeat code. The while loop runs the code inside, while the condition is True. In this example the condition is the variable price. That means as long as price contains any information at all, it will be considered True, and the loop will run again.

4 / 6

Inside our loop we have two lines of code. The first line converts the input from a str to a float, and adds it to the variable total. This means the value of total is increased by the amount of price. We use float() to convert the input to a float, which is a number data type for working with decimals.

5 / 6

Then we ask the user for the next price. Everytime through the loop, the variable price is checked. If it contains something, it runs the loop again. When the user stops entering prices and just presses enter, the variable price won't have anything in it, it is empty. Python treats an empty variable as False so the loop stops immediately.

6 / 6

Then the code after the loop runs.
