While n is less than...

Okay coder, we've seen lots of uses for a while loop, but there are many other ways to set the condition. Let's use an int variable to control interations through a loop. Each time through a loop is called an iteration. Let's set a variable n and set the condition for our loop to be that n is less than 10. On each iteration through the loop, let's increment n by one. In python code that looks like this

n = 0
while n < 10:
  print(f"n is {n}")
  n += 1
print(f"The loop is finished. n is {n}")

Notice that if we never increment n we would have an infinite loop!
