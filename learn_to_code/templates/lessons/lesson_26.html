This or that?

Okay coder, we've seen how to check if both conditions are True in the if. But what if we only care if one of the conditions is True? Well, we can just use or. Check it out with another video game example!

lives_left = 0
bonus_life_available = True
if lives_left > 0 or bonus_life_available:
  print("You can keep playing!")
else:
  print("Game over!")

By combining the or with the if statement you can make sure the code in the if statement executes if only one (or both!) of the conditions is True. Try this: Experiement with a differnet number for lives_left and set bonus_life_available to False.
