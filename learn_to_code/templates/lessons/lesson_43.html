While n is greater than...

Okay coder, let's use an int variable to control interations through a loop another way. Let's set a variable n and set the condition for our loop to be that n is greater than 0. On each iteration through the loop, let's decrement n by 5. In python code that looks like this

n = 100
while n > 0:
  print(f"n is {n}")
  n -= 5
print(f"The loop is finished. n is {n}")

Notice that if we never decrement n we would have an infinite loop!
