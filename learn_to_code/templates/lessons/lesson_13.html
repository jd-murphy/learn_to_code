four = 14 + 93 ?!?!?!

Okay coder, we've got to try something a little crazy here to prove a point. Let's say we want to name our variable four. Technically, it is totally legal in python to name a variable four. It might even seem like a good idea if, for example, we need to save the answer to the question 2 + 2. Like this

four = 2 + 2
print(four)

Seems good right? And even if we do this

four = 16 / 4
print(four)

But be careful, because this is just as legal but can get very confusing very quickly.

four = 983 / 23.4
print(four)
