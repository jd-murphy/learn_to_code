Comparing variables

Okay coder, we've seen how do a lot of comparison tests but we have only stored the comparison result in a variable. We can do much cooler things. Check this out. Say we are coding a video game. We might give the player 3 lives before they have to restart a level. We can set a variable to keep track of how many lives they have left.

lives_left = 3
print(lives_left)
lives_left -= 1
print(lives_left > 0)
lives_left -= 1
print(lives_left > 0)
lives_left -= 1
print(lives_left > 0)

You can see that every time we take away a life, we check to see if the player has enough lives left to continue playing. As long as lives_left is greater than 0, we let the player continue.
