Let's get fancy with print()

Okay coder, we've seen how to print out the results, but we can add a little more flexibility to our print() statements. Over the next few lessons we will learn a few different ways to print() to the console.

result = 5 * 16
print("The result is", result)

Arguments are whatever we pass into a function like print(). We use a comma to separate our arguments, in this case, our string and our variable. Python will print() out the arguments (our string "The result is" and our variable result) separated with a space.
