Okay coder, we've seen how to do math in variables, but we need to have a talk about something that might get confusing. How do we decide what to name our variables? We have a lot of options. We can use letters and numbers and even the underscore _ !
We also need to avoid special words that python depends on. They are called keywords and we will talk about them more later. But let's talk about confusing names. For example, when we are doing math we might want to name our variable answer or result... But when we are asking someone's name it makes more sense to call our variable name, right?

answer = 5 + 416 / 7
print(answer)
name = "Thomas"
print(name)

We can even use a single letter like x, or i. But it's usually a really good idea to make sure the name of the variable makes sense. We should avoid using confusing variable names, as we'll see in the next lesson.
