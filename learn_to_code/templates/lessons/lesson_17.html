Which is more?

Okay coder, we've seen how to check if two things are the same or not with is and ==. Now lets see if something is greater than another thing. Take 3 and 8 for example. Is 3 greater than 8? Python will say that is False. But, what if we switch them around, and ask if 8 is greater than 3? 8 is indeed greater than 3, so python will say this is True. Let's use > to check if the number on the left is greater than the number on the right. Like this

result = 5 > 16
print(result)
result = 9 > 3
print(result)

The number on the left will be compared with the number on the right. If the left side is greater than the right, the answer will be True. Otherwise the answer will be False.
