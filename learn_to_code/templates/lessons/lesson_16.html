Is IS Equals?

Okay coder, we've seen how see if something is something, but we can also check if something equals something. They aren't exactly the same thing... But they are very similar. Let's learn about is and equals. Equals is written in python as ==. We use one equals sign = to put information into a variable. We use two equal signs == to check if two things are equal.

result = 4 is 4
print(result)
result = 4 == 4
print(result)

You can also find out something is not, or is not equal to, another thing.

result = 'cat' is 4
print(result)
result = 92.123 == 8
print(result)

Later on we will see some situations where is and == aren't the same thing...
