Add up HOW MANY numbers?!?!

Okay coder, maybe we need to write code for a cash register or an online shopping application. We don't know how many items will be purchased. We don't want to limit our user to a specific number of items, and every user can purchase a different number of items. We need a new tool! Let's learn about loops! There are different kinds of loops, but we'll start with the while loop. It's perfect for a cash register or shopping application. We will continue to ask for user input, while the user has something to enter. Loops need a colon : and indentation, just like if statements! When we work with numbers that have decimals (like money), we use a different data type than int. Think of using int for counting numbers: 1, 2, 3, 4... When you need decimals like 23.54 or 19.99, you use the data type float. It works just like int. Enter as many prices as you want, and press enter each time. When you are finished, press enter to run the rest of the code.

total = 0.0
price = input("Enter the price of the item: ")
while price:
  total += float(price)
  price = input("Enter the price of the next item: ")

print(f"Your total is ${total}")
