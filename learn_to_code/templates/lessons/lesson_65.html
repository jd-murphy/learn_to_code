Functions for the win!

Okay coder, we've written a few functions, but we need more practice. Let's use functions to get numbers from the user and multiply them all together. Let's write two functions, one to get the numbers, and one to do the math. In order for our functions to share the result of their processing, we need them to return the result. We use the return keyword. To accept a return value from a function, we assign it to a variable, or print() it out.

def get_input():
  print("(Press enter twice to quit.) ")
  return input("Enter your number: ")

def multiply_all(num_list):
  total = 1
  for num in num_list:
    total *= num
  return total

nums = []
n = get_input()
while n:
  nums.append(int(n))
  n = get_input()

print(multiply_all(nums))
