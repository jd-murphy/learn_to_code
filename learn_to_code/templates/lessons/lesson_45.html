Lesson 45

Fancy for loops

Okay coder, let's look at how for loops work with range(). A for loop completes one iteration of the loop for each thing. A range() gives us a sequence of numbers. When we use a for loop with a range() we get one iteration for each number in the range. If we give range() one argument, it gives us a sequence from 0 up to (but not including) the number we give it.

for i in range(10):
  print(f"Once for i when i is {i}")

If we give range() two arguments it gives us a sequence from the first number up to (but not including) the second number.

for i in range(10, 20):
  print(f"Once for i when i is {i}")

If we give range() three arguments it gives us a sequence from the first number up to (but not including) the second number in steps the size of the third number!

for i in range(30, 70, 5):
  print(f"Once for i when i is {i}")
