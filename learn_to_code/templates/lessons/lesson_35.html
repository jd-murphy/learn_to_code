Numbers are not strings!

Okay coder, we've seen how to take in input and work with text. But say we want to take in some numbers, like a cash register's software would, or on online banking application. We have to be aware that different kinds of data mean different things to python. For example, you can type in numbers and strings. When python reads them in through the input() function, python converts everything to a string. This is our first encounter with data types. In python, strings have a data type of str. To python, everything in "double quotes" or 'single quotes' is considered a string of type str. And, for example, you can't do math with strings. If you give python a number in the input() function, even though you typed a number, to python, it is now a string. To do math with our number it must first be converted from a string back to a number data type: either int or float. Let's learn how to take input and turn a str into an int!

input_num = input("Enter a number please: ")
input_num = int(input_num)
plus_five = input_num + 5
print(f"{input_num} plus five is {plus_five}")

Voila! We can work with data no matter what type it is!
