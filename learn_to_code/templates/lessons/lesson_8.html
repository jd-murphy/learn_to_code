Variables are awesome!

Okay coder, we've seen how to do some math, but most of the time we won't be doing math inside a print() statement. How can we do math outside a print() statement and still see the result? Well my friend, the magic is in variables! Python uses variables to store information. Try this

result = 5 * 16
print(result)

Don't use quotes " ", just print out the variable named result. Look in the console and you'll see the magic!
