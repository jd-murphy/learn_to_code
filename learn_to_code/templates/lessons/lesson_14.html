Lesson 14

result = result + 1

Okay coder, we've seen how to use a variable to store information and how to get information back out of it. We can even use a variable to change itself like this

result = 2
result = result + 1

But these kinds of operations are a special case, and so we have another special way to write it. If you need to change the variable by a number, you can use this short notation.

result += 1
result -= 2
result /= 3
result *= 0.5

These can be confusing, but they are just a short way to use the variable to change itself. You can think of result += 1 as "increase the variable result by 1" and think of result -= 2 as "decrease the variable result by 2" The most common way you will see this is the += 1 scenario and sometimes the -= 1 scenario.

Your variable must have a value BEFORE you do this or you will get an error!
