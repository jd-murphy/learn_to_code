A different kind of loop

Okay coder, let's learn a new kind of loop. We are going to work with for loops now. In order to best understand a for loop, we need to introduce another built in python function. Say hello to range()! We often will see range() used together with for loops in python. A range() let's you create a sequence of numbers very easily. For example, you can print() a sequence of numbers very easily with a range() and a for loop, like this

for i in range(100):
  print(i)

You'll notice the range() function goes from 0 up to one less than the number you pass in as an argument. In the above example our range() went from 0 to 99 because we passed in 100. A lot of things in python start at 0 instead of 1. This might seem strange but I promise you'll get used to it. If you don't want a range() to start at 0 you can give it two arguments, the first is where it starts and the second is the number it goes up to (this number is not included in the range()).

for i in range(15,31):
  print(i)
