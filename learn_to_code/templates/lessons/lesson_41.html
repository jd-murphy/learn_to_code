Break out!

Okay coder, we've seen why we avoid infinite loops, but that doesn't mean we can't use while True. We need to have a way to end our loops at some point. Let's use a video game example to see how we could use while True and break out of the loop. When you code a video game, you want the game to run until the user is done playing. We don't know when that will be, so we'll use a while loop and have the condition set to while True. When the user is done, we'll use a python keyword called break. This ends the loop. Notice the "\n" in our string. This tells python to start a new line.

while True:
  # this is a comment
  # let's pretend this is our game code
  print("Starting level...\nPlaying level...\nFinished.")
  # at the end of each level ask user
  play_again = input("Do you want to keep playing? (y/n) ")
  if play_again.lower() == 'n':
    break
print("Your progress was saved. Thanks for playing!")

As you can see, it might be good to set a loop that has no specified end, and let our user choose when to end the program. This isn't an infinite loop because we allow the user to break out of the loop when they're done playing.
