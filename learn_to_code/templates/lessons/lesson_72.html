Making a list of primes

def is_prime(num):
   for i in range(2, num):
     if num % i == 0:
       return False
   return True
def create_prime_list(count):
   primes = []
   for i in range(count):
     primes.append(2+i)
   for j in range(len(primes)):
     while not is_prime(primes[j]):
       for k in range(j,len(primes)):
         primes[k] +=1
  return primes

We will write two functions, the first will be a helper function to check if a number is prime. The second will create the list of primes. It will ask the helper function for assistance.

3 / 15

First we define is_prime() with the def keyword. We give it a parameter named num to hold our argument when we call the function. This will hold the number we want to check.

4 / 15

Inside of is_prime() should be very familiar. We loop through a range() from 2 up to (but not including) the number we want to check.

5 / 15

For each number in our range() we ask if num % i == 0. We are asking if the number is evenly divisible by each number. We use the modulo % operator. If this is True we return False because this number is not prime.

6 / 15

If we get all the way through the numbers between 2 and our number and have not found a number that divides evenly, we know the number has no factors aside from 1 and itself. Therefore, the number is prime. We return True.

7 / 15

Next we define create_prime_list() with the def keyword and define the parameter count. This will take user input as an argument. This number will tell us how many primes we need to put in our list.

8 / 15

We declare and initialize an empty list: primes.

9 / 15

We loop through a range() the size of the parameter count which holds an argument given by our user. We add a placeholder value into our list primes, starting with 2 and increasing it by one each number through the range. If you print() out primes at this point you will see it is [2,3,4,5,6... (containting count values)]

10 / 15

Now we use a range() the length of primes to loop through primes. We use j as an index to access each element in primes like so: primes[j]

11 / 15

We create a while loop to check each number in primes: the condition - while primes[j] is not prime is evaluated by calling the is_prime() function we wrote earlier. It returns True if the number is prime and False if the number is not prime. We loop until the number is prime. The not keyword negates the return value from is_prime() so our loop continues until is_prime() returns True.

12 / 15

Inside this while loop (while the current number is not prime) we use a range() to loop through and increase the current number and every future number in the list. We check the current number again to see if it is prime yet.

13 / 15

The above loops repeat until every number in primes is truly a prime number. We can now return our list primes.
