Lesson 49

Loops and lists

Okay coder, we can use our for loops to iterate through a list. It's just as easy as looping through a range().

names = ['Thomas', 'Rebekkah', 'Rachel', 'Aaron', 'Andre']
print(names)
print('--------')
for name in names:
  print(name)

You can see the difference between simply calling print() on the list and using a for loop to iterate through each of the names in the list and print() them out individually. If we need access to each individual element in a list it's best we use a for loop.
