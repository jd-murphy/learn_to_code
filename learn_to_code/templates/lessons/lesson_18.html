Lesson 18

Which is less?

Okay coder, we've seen how to check if something is greater than something, but what about checking to see if a thing is less that another thing? We can ask if the thing on the left is less than the thing on the right. We will use < to ask python to check for less than. Is 5 less than 16? Yes. Python will say this is True. Is 16 less than 5? Nope. So python says False.

result = 5 < 16
print(result)
result = 16 < 5
print(result)

A special thing to watch out for with > and < is to realize that when something is equal, it is not less than or greater than. So 5 and 5 can be compared with > or < but we will always get False.

result = 5 < 5
print(result)
result = 5 > 5
print(result)
