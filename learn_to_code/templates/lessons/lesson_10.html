name = "Python Master"

Okay coder, we've seen how to put things in variables. But it isn't very useful to put something in a varibale and then not do anything to it! Let's learn how to change a variable! You can change a variable the same way you set a variable. It's that simple. We might set name = "Python Beginner" but soon we can change it to name = "Python Student" and soon enough we'll have to change it to name = "Python Master"! Make sure each assignment is on its own line. Try this out!

name = "Python Beginner"
print(name)
name = "Python Student"
print(name)
name = "Python Master"
print(name)

It is incredibly powerful to work with variables. We are going to be learning more ways to work with variables soon. It's going to get really fun!
