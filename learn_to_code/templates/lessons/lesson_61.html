Santa's list code review

Okay coder, That was a lot, let's review the code for Santa's list from the last lesson! First, we declare and initialize two empty lists. One for nice children, and one for naughty children. Then we start getting input() from the user. We ask for a child's name and which list they should go on. We start a while loop to continue asking for names until the user is finished. The condition controlling the loop is that the user must enter some value for the child's name and the destination list. If the variables are empty, the loop will exit. We convert the string in the variable destination to lowercase and compare it to the string 'nice'. If the comparison is True, we will .append() the child to the nice_list. If the child is not destined for nice_list, we check if they are headed to the naughty_list in the elif clause. We convert the string in the variable destination to lowercase and compare it to the string 'naughty'. If the comparison is True, we will .append() the child to the naughty_list. If the user typed a string that doesn't match 'naughty' or 'nice', we ask them to enter the child's information again. We repeat the process until the user is finished entering names. When the user is done they can press enter twice without entering a child's info, and the program will exit the loop and print out the resulting lists.

nice_list = []
naughty_list = []
child = input("Add a child: (press enter twice to quit) ")
destination = input("Which list? ")
while child and destination:
  if destination.lower() == 'nice':
    nice_list.append(child)
  elif destination.lower() == 'naughty':
    naughty_list.append(child)
  else:
    print("Try adding the child again.")
  child = input("Add a child: (press enter twice to quit) ")
  destination = input("Which list? ")
print("nice_list")
print(nice_list)
print("naughty_list")
print(naughty_list)
