Lesson 27

Too many ifs?

Okay coder, we've seen how to check conditions with if and else, and we can now make some really complex logic. But there is another tool we can use. What about checking if something is True, but then checking something else only if the first condition is False? Well let's learn how to use elif!

score = 56
if score > 100:
print("Wow! You scored more than 100 points!")
elif score > 50:
print("Great job! You scored more than 50 points!")
else:
print("Keep trying! You scored less than 50 points!")

Try setting score to different numbers so that you get each result to print out.

Tip: You can add as many elif clauses as you want, but you can only have one else for each if. Else is not required however, else must always be after any elifs that you have.
