How long is that list?

Okay coder, we've seen how to get items from lists either by using a for loop or by accessing the elements by index directly. How do we know how long a list is? This is important because we can get errors if we try to access an element at an index that is larger than the length of the list. Remember lists start at index 0 so the number of elements in a list is NOT the same as the index of the last element. Let's use len() to see how long a list is.

scores = [199022, 205600, 190406, 300350, 872954]
print(len(scores))

The indexes of the elements in this list are 0, 1, 2, 3, 4

scores = [199022, 205600, 190406, 300350, 872954]
print(len(scores))
print(scores[0])
print(scores[1])
print(scores[2])
print(scores[3])
print(scores[4])

The last item is at index 4 and the length of the list is 5!
