Lesson 51

Accessing elements

Okay coder, we've seen how to use a for loop to get each item in a list. What if we only want a specific item though? Well, in python we can use an index to get specific items out of a list. All lists start at index 0, just like our range() did when we only gave it one number. We can use square brackets [ ] to access an element in a list by index. To get the first element from a list, it looks like this

my_list = [19, 22, 28.232001, 19992, 24, 14, 0, 3.1]
item = my_list[0]
print(item)

Grabbing the second item (at index 1) would look like this

names = ['Sara', 'Luke', 'Savannah', 'Kyle']
name = names[1]
print(name)
