Lesson 57

Even or odd?

Okay coder, let's use what we learned to check if each element in a list is even or odd. We need one more tool to do this. Let's learn about the operator modulo %. It's a math operator just like +, -, *, or /. The modulo operator % uses division / to find the remainder of division between two numbers. If we divide 10 by 2 the answer is 5, and there is nothing left over. The remainder is 0. If we divide 10 by 3 the answer is 3 with a remainder of 1. We use modulo % to find remainders. This works nicely for checking if a number is even or odd. Any even number divided by 2 is 0. We ask python if some number % 2 equals 0. The answer is True if the number is even, and False if the number is odd.

nums = [3, 5, 2, 6, 9, 19, 34, 23, 6, 13, 930, 234, 123, 643, 5, 353, 7564, 99, 102]
for i in range(len(nums)):
  if nums[i] % 2 == 0:
    nums[i] = 'Even'
  else:
    nums[i] = 'Odd'
print(nums)

We are using modulo % to check if each number is divisible by 2. If so, that means the number is even. We replace every even number with the string 'Even'. Everything else is not even, and therefore odd. We replace every number that is not even with 'Odd'.
