A second look at add_tax

Okay coder, let's take a close look at our function add_tax() from the previous lesson.

def add_tax(total):
  tax_rate = 8.25 / 100
  tax = total * tax_rate
  return (total + tax)

We define the function using the def keyword and tell python the name of this function will be add_tax. Inside the required parentheses, we code a parameter named total to hold an argument when we call the function. We set a variable called tax_rate to be a float representing the percentage of tax for the transaction (8.25%) which we divide by 100 to get a decimal value for the tax percentage. Then we multiply the correct decimal tax rate by the parameter total, which will hold the value we pass in when we call the function. After storing the calculated tax, we have to add it to the total and return the calculated total which now includes the tax. We group total + tax with parentheses to make it clearer what is happenening. In this particaluar case it does not change the outcome of the calculation.
