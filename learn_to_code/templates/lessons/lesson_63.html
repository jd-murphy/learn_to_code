Let's keep the momentum up. We're going to learn about functions. We've already used some of python's built in functions but now we're going to learn how to write our own functions. Basically, a function is some code we package together and give a name. When we want to use the code, we call the function, using the name we gave it.

def my_first_function():
  print("Hello, from my_first_function()!")

my_first_function()

First, we define the function using the keyword def. We put parenthesis () and a colon : at the end of the name. We use indentation (like if statements or loops) to format the code we need the function to run. We call the function by the name we gave it when we need to use it. When you call the function put parenthesis () after the name!
