print() it again!

Okay coder, we've seen how print() by separating arguments with a comma, but there are more ways to print() out our information. Another option is using .format()

result = 5 * 16
print("The answer is {}".format(result))

name = "Sarah"
age = 23
print("{name} is {age} years old".format(name=name, age=age))

You can pass a single argument to .format() or you can pass more than one! You can see from the example above that if we only use one argument we use empty curly braces {} in our string. If you want to pass more than one arguments you can pass them as keywords argumentsas in the second example. Make sure to name the arguments in the curly braces in your string! Look carefully at the examples to see the difference.
