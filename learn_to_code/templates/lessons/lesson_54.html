Let's change that list!

Okay coder, we've seen how to access the elements in a list, and check how long a list is with len() to avoid errors. Now we know enough to start changing our list and the elements in it! Let's loop through a list and change each element!

nums = [1, 4, 3, 6, 7, 2]
for n in range(len(nums)):
  nums[n] = 5
print(nums)

That might be a lot to take in, so let's talk through each step. We start with a list. We find the length of the list with len(). The length is 6 so we use 6 to create a range() from 0 to 5. Then we loop through the sequence of numbers that range() gives us: 0, 1, 2, 3, 4, 5. Each iteration through the loop, n is going to be the next number from the range(). We use n as the index to access each element in the list nums. Because we used the length of nums and range() to control our loop, we don't need to worry about index errors. They both start at index 0. Each time through the loop we change the current element to 5.
