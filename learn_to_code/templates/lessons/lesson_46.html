Backwards too?

Okay coder, let's look at some other ways to work with for loops and range(). We know that if we give range() three arguments it gives us a sequence from the first number up to (but not including) the second number in steps the size of the third number! We can give a negative number to walk through the range() backwards. We need to make sure the first number is larger than the second number and the third number must be negative.

# count backwards from 100 by 5s
for i in range(100, 50, -5):
  print(f"Once for i when i is {i}")
# count backwards from 21 by 3s
for i in range(21, 0, -3):
  print(f"Once for i when i is {i}")
# count backwards from 1000 by 100s
for i in range(1000, 0, -100):
  print(f"Once for i when i is {i}")
