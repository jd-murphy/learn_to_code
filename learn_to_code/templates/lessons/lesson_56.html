Double those elements!

Okay coder, let's use what we learned to double each element in a list.

nums = [200, 100, 600, 900, 500]
for i in range(len(nums)):
  nums[i] *= 2
print(nums)

We are using the same process as in the last few lessons. We find the length of the list with len(). The length is 5 so we use 5 to create a range() from 0 to 4. Then we loop through the sequence of numbers that range() gives us: 0, 1, 2, 3, 4. Each iteration through the loop, i is going to be the next number from the range(). We use i as the index to access each element in the list nums. Because we used the length of nums and range() to control our loop, we don't need to worry about index errors. They both start at index 0. For each item in the list we double it!
