Welcome to Python!

Okay coder, Welcome to your second coding lesson!

Python is a very powerful programming language. But it's easy to learn! In the editor you can type your Python code. The console will print() out the results. The parentheses ( ) and quotes " " are very important to Python. Go ahead and type whatever you want in between the parentheses (and inside the quotes!) like this

print("We can print anything! Python rocks!")

Don't press the Enter key when you're inside the print statement, Python needs everything inside the print statement to be on the same line. Click the green button above the editor and run your code when you're ready!

You should see your results printed out in the console!
